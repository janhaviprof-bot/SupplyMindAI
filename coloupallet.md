# SupplyMind AI Color Palette

UI color palette derived from the SupplyMind logo (black, electric blue, cyan, white).

## Core Palette

| Token | Hex | Usage |
|-------|-----|-------|
| Primary (electric blue) | `#0d6efd` | Primary buttons, main links, key actions |
| Primary light | `#4dabf7` | Hover states, lighter accents |
| Primary dark | `#0958d9` | Active/pressed states |
| Cyan accent | `#22b8cf` | Secondary accents, highlights |
| Cyan light | `#63e6e2` | Subtle highlights, borders |
| Dark blue | `#1e3a5f` | Headings, primary text on light backgrounds |
| Charcoal | `#212529` | Body text, headers |
| Gray | `#6c757d` | Muted text, placeholders |
| Light gray | `#e9ecef` | Borders, dividers, subtle backgrounds |
| Off-white | `#f8fafc` | Page background, card backgrounds |
| White | `#ffffff` | Cards, inputs, high-contrast surfaces |
| Black | `#0a0a0a` | Logo background, dark mode base |

## Semantic Colors

| Role | Hex | Example |
|------|-----|---------|
| Success | `#198754` | On-time shipments, positive metrics |
| Warning | `#ffc107` | Delayed shipments, attention needed |
| Danger | `#dc3545` | Critical shipments, errors |
| Info | `#22b8cf` | Informational tips, neutral accents |

## Usage Guidelines

- **Primary actions**: `#0d6efd`
- **Headers / navigation**: `#1e3a5f` or `#212529`
- **Page background**: `#f8fafc`
- **Card backgrounds**: `#ffffff`
- **Accents / badges**: `#22b8cf` for AI-related elements
- **Borders / dividers**: `#e9ecef`
